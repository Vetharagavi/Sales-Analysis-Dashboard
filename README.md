 Sales Analysis Dashboard – Power BI Project

Project Overview
This project focuses on building an interactive Sales Analysis Dashboard using Microsoft Power BI. The goal is to analyze sales performance across different regions, categories, and products, and to provide meaningful insights for decision-making.


 Objectives
- Analyze overall sales performance
- Compare sales against targets
- Identify top-performing products and categories
- Understand regional sales distribution
- Build an interactive dashboard using filters and visuals


Dataset Description
The dataset used for this project contains the following fields:
- City
- State
- Postal Code
- Region
- Product ID
- Category
- Sub-Category
- Product Name
- Sales
- Target
